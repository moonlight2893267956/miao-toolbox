## Spring Boot 4.x / Spring Security 6 SSE 最佳实践
- SseEmitter 异步回写触发 Tomcat async dispatch → 重走 FilterChain
- OncePerRequestFilter 默认跳过 ASYNC，但 AuthorizationFilter 不跳过
- 解决：dispatcherTypeMatchers(DispatcherType.ASYNC).permitAll()
- 这是 Spring Security 官方推荐处理 SSE 的方式
- 在 SecurityContext 中无效（ThreadLocal），SecurityContext 捕获方案是 workaround
