### Source intake

### Initial observations
- Two distinct user roles: 普通用户 (use tools) + 管理员 (monitor & manage)
- Core surfaces identified: Login, Tool list, Tool operation, Admin dashboard
- PRD defines two key journeys: UJ-1 (user uses a tool), UJ-2 (admin monitors anomalies)
- v1 is a security-first skeleton — the UX needs to make security feel invisible, not heavy
